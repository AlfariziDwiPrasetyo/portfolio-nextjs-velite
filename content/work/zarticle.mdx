# Zarticle

Zarticle is a modern and efficient article platform designed to help users create, manage, and share articles effortlessly. Whether you're a blogger, journalist, or content creator, Zarticle provides the tools you need to streamline the article publishing process.

## Features

- **Article Creation**: Users can write and publish articles with a rich text editor, including support for images, links, and formatting options.
- **Category**: Organize articles with categories and tags for better content discovery.
- **User Authentication**: Secure login and user management to allow authors to manage their articles.
- **SEO Optimization**: Built-in SEO tools to help improve article visibility on search engines.
- **Comment System**: Engage with readers through a built-in commenting feature.

## Built With

Zarticle is built using modern web technologies to ensure a seamless and scalable experience:

- **Frontend**: [Next.js](https://nextjs.org/) for a fast, dynamic, and SEO-friendly user interface.
- **Database**: [PostgreSQL (Neon)](https://https://neon.tech/) for flexible data storage.
- **Authentication**: [Auth.js](https://authjs.dev/) for secure and easy-to-use user authentication and management.
- **UI Framework**: [Tailwind CSS](https://tailwindcss.com/) for a sleek and responsive UI.
- **Validation**: [Zod](https://zod.dev/) for robust data validation and type safety.

## Main Programming Language

The primary programming language used in Zarticle is TypeScript, ensuring type safety and maintainability throughout the project.

## Core Features

- **Article Management**: Users can create, edit, and delete articles.
- **Rich Text Editor**: A user-friendly editor with markdown support.
- **Content Categorization**: Organize content with categories and tags.
- **User Profiles**: Allow users to have their own profiles with their published articles.
- **Commenting System**: Users can leave feedback and engage in discussions.

Zarticle is built for writers, content creators, and publishers who want a simple yet powerful platform for managing and sharing their articles efficiently.
