# Arukaifu

Arukaifu is a dedicated platform designed to help users remember and archive important information that tends to slip my mind.

## Features

- **Memory Aid**: Arukaifu serves as a reliable memory aid, ensuring users don't forget crucial information.
- **Archive**: Users can archive important data for future reference, keeping everything organized and easily accessible.

## Built With

Arukaifu is built with the following technologies:

- **Frontend**: [Next.js](https://nextjs.org/) for building dynamic web applications with React and TypeScript.
- **Database**: [Supabase](https://supabase.io/) for managing data storage and authentication.
- **Image Storage**: [Cloudinary](https://cloudinary.com/) for storing and managing archived images.
- **UI Framework**: [Daisy UI](https://daisyui.com/) for building responsive and accessible user interfaces.

## Main Programming Language

The main programming language used in Arukaifu is TypeScript.

## Features

- **WYSIWYG Note Archive**: Users can archive notes with a What-You-See-Is-What-You-Get (WYSIWYG) editor for enhanced formatting.
- **Image Archive with Cloudinary**: Arukaifu allows users to archive images, which are stored and managed using Cloudinary.
- **Task Archive**: Users can also archive tasks to keep track of important to-dos.

## Authentication

Authentication in Arukaifu is handled through Supabase Auth, providing secure user authentication and management.
